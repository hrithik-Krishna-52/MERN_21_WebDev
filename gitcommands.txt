Follow the following commands to commit code on github :-

1. git init : initialize an empty .git folder in the current working directory.

2. git add . : stage al the files in the current working directory
                (to indicate which files should be added to the .git folder).

3. git commit -m "commit message" : to commit the files into the .git folder

4. git branch -m main : to create a branch in the repository 
(branch meaning sub-folder)

5. git remote add origin <github repository url> : to link the local respository
 to the remote repository (git folder in the github server)

6. git push -u origin main : to push teh local respository to the remote repository.

After modifying the code
repeat the stages 2,3 and 6
